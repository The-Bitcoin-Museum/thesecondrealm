INTRODUCTION TO GUIDED TOURS

Beyond freely wandering through its data landscapes, the Second Realm offers the option to follow a guided tour.

A guided tour is an ordered collection of short stories tied to specific points of interest within a realm. This collection forms a circuit, serving as an Ariadne’s thread to guide explorers through their immersive journey.

DESCRIPTOR FILES FORMAT

A guided tour is defined by a descriptor file in JSON format, associated with the realm by its builder during configuration. An example of a descriptor file can be found here.

The structure of all descriptor files follows this format:

version: The version of the Descriptor File format. Value should be 0.1.0

type: The type of descriptor file. Value should always be "guided_tour"

title: The name of the Guided Tour. Value should be a string.

coords_type: The coordinate system used in the file. The value should be either "data_series" or "block_height".

  • data_series: Coordinates are expressed as an array of three numerical values [X, Y, Z] representing a point in the data space, dependent on the metrics selected for each axis.
  • block_height: Coordinates are expressed as a single integer value representing the height of a block.

initial_pos: The initial position of the explorer. The format depends on the "coords_type" value.

stages: An array of points of interest (stages). Each stage is an object with the following structure:

  • coords: The coordinates of the point of interest. The format depends on the "coords_type" value.
  • title: The name or title of the stage.
  • desc: The content associated with the stage.

TIPS

Using data values as coordinates allows points of interest to be defined independently of block positions, making it easier to fine-tune their placement for optimal visibility. However, this approach requires that the guided tour be used with realms configured with the same metrics for all three axes.

The guided tour is not just about the story—it’s also about the journey. The order of the points of interest significantly impacts the experience.
