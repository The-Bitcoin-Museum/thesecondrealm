## Contributing

Many improvements are needed:
 - stability of the app
 - performances (compression artefacts, etc)
 - new features
 - ...

Help us to build The Second Realm!

1. Fork it
2. Download the archive of static files from https://thesecondrealm.is/static/thesecondream_static.zip and extract its content into the 'static' subdirectory of your project directory
3. Create your feature branch (`git checkout -b my-new-feature`)
4. Commit your changes (`git commit -am 'Add some feature'`)
5. Push to the branch (`git push origin my-new-feature`)
6. Create new Pull Request
